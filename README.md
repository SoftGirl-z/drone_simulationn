## üìå Project Overview

This project simulates a quadcopter drone in a 3D environment using Unity.  
It demonstrates how drone flight mechanics, user input, physics, and UI systems can work together in a modular simulation.

The simulation includes manual flight control, propeller dynamics, collision handling, and a heads-up display (HUD) for real-time feedback.
